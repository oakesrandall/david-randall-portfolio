ABOUT THE PROJECT

Foodr was built with the idea that people would be more likely to try restaurants and cuisine outside their comfort zone if they were only shown photos of the food. This app determines the location of the user using their device's location.

This app makes an API call to Google Places and pulls in a group of nearby restaurants. Then, the app makes API calls for detailed information on each nearby restaurant. That detailed information contains photos submitted to Google Places of the food at those restaurants.

This app was built as a group project using Angular 4 and Firebase.
